Bicycle API

Welcome to the Bicycle API. Feel free to view, create and manage all of your bicycles here. Use the routes below.

GET /api/bikes - Get all bikes

POST /api/bikes - Create a new bike

  • name
  • email
  • password

GET /api/bikes/:id - Get bike

PUT /api/bikes/:id - Edit a bike

  • name
  • email

DELETE /api/bikes/:id - Delete a bike
